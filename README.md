## WeCAR_udev_rules
Udev Rules for WeCAR

# Description
Make Udev Rules for RPLidar A2, VESC, Razor IMU 9DOF, WebCAM

# How to use

```bash
$ git clone https://github.com/WeGo-Robotics/WeCAR_udev_rules
$ cd WeCAR_udev_rules
$ chmod +x create_serial_rules_wecar.sh
$ ./create_serial_rules_wecar.sh
```
